## 🇺🇸 ENG

### Clone😎

```
git clone https://github.com/jovimoura/igniteLab-platform
```

### Access the folder🤓

```
cd <folder>
```
### Install dependencies🤠
```
npm install
```
### Start application🤩
```
npm run dev
```
### default port:🤗

```
http://localhost:3000/
```
